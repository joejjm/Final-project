# Video Image Analysis AI Project

This project analyzes video files to detect changes using AI techniques.

## Structure
- `src/`: Source code for video/image analysis
- `data/`: Place your video files here
- `notebooks/`: Jupyter notebooks for experiments

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the sample script in `src/` to test video analysis

## Requirements
- Python 3.8+
- OpenCV, scikit-image, numpy, tensorflow (or torch)
